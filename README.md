# Online Harassment Detection

## Overview
This is a Final Year B.Tech AI & ML project developed to detect online harassment and abusive language using Natural Language Processing (NLP) and Machine Learning.

## Features
- Detects online harassment and abusive language
- User Login and Registration
- Service Provider Dashboard
- Harassment Prediction
- Database Integration

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- MySQL
- NLP
- Machine Learning

## Project Structure
- Database
- Remote_User
- Service_Provider
- Template
- fadohs

## How to Run
1. Install Python and Django.
2. Import the MySQL database (`fadohs.sql`).
3. open the project folder in Terminal.
4. Run:
     python manage.py runserver
5. Open your browser and visit:
       http://127.0.0.1:8000/
